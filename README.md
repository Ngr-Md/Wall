# سامانه ثبت آگهی خرید و فروش کالای دست دوم: Wall

نام های اعضای گروه:
ایلیا حامدی بصیر،
نگار مدقق

یک سامانه Client-Server برای ثبت و مدیریت آگهی‌های خرید و فروش کالای دست دوم شامل:

- **Backend**: Java 26 + Spring Boot 3 (REST API + JWT + Spring Data JPA)
- **Frontend**: JavaFX 21 (FXML + CSS)
- **ذخیره‌سازی پایدار**: SQLite

---

## نحوه اجرا

### 1) اجرای Backend

```bash
cd backend
mvn spring-boot:run
```
(به طور جایگزین، میتوان از داخل IDE و فایل WallApplication بک اند را اجرا کرد)

سرور روی `http://localhost:8080` بالا می‌آید و فایل دیتابیس `wall.db` به‌صورت خودکار ساخته می‌شود.

بررسی سلامت سرور:

```bash
curl http://localhost:8080/api/health
# {"status":"UP"}
```

### 2) اجرای Frontend

در یک ترمینال جداگانه (در حالی که Backend اجراست):

```bash
cd frontend
mvn javafx:run
```
(به طور جایگزین، میتوان از داخل IDE و با پنل maven فرانت اند را اجرا کرد)

## قابلیت‌های سامانه

### کاربر عادی
- ثبت‌نام و ورود
- ثبت آگهی با عنوان، توضیحات، قیمت، دسته‌بندی، شهر و حداکثر ۵ تصویر
- ویرایش و حذف آگهی خود (پس از ویرایش، دوباره در انتظار تایید قرار می‌گیرد)
- اعلام «فروخته شد» برای آگهی تاییدشده
- جستجوی پیشرفته: متن، دسته‌بندی، شهر، محدوده قیمت، مرتب‌سازی (جدیدترین/قدیمی‌ترین/ارزان‌ترین/گران‌ترین)
- مشاهده جزئیات آگهی و تصاویر
- گفتگو با فروشنده درباره هر آگهی
- نشان کردن آگهی‌ها
- امتیازدهی ۱ تا ۵ به فروشنده همراه با نظر

### ادمین
- مشاهده فهرست آگهی‌های در انتظار تایید
- تایید یا رد آگهی (رد با ذکر دلیل — دلیل به کاربر نمایش داده می‌شود)

## نکات فنی

- احراز هویت با **JWT** انجام می‌شود؛ توکن در هدر `Authorization: Bearer <token>` ارسال می‌شود.
- رمز عبور کاربران با **BCrypt** هش می‌شود.
- تصاویر آگهی به‌صورت **Base64** در دیتابیس ذخیره می‌شوند تا وابستگی به فایل‌سیستم نباشد.
- آدرس سرور در کلاینت از `frontend/.../config/ApiConfig.java` قابل تغییر است.
- مستندات کامل API در [docs/API.md](docs/API.md) موجود است.
